## 🚀 虚拟机创建脚本 (2个)

### 6. create-windows-vm.sh
- **用途**: Windows 虚拟机启动脚本
- **功能**:
  - 自动创建磁盘镜像
  - 完整的反虚拟化配置
  - 自定义硬件参数
  - 端口转发 (RDP 3389)
- **使用**: `./create-windows-vm.sh`

### 7. create-linux-vm.sh
- **用途**: Linux 虚拟机启动脚本
- **功能**:
  - 自动创建磁盘镜像
  - 反虚拟化配置
  - 随机生成序列号
  - 端口转发 (SSH 2222)
- **使用**: `./create-linux-vm.sh`

## 🎲 硬件参数生成工具 (2个)

### 8. generate-hardware-params.sh
- **用途**: Linux/macOS 硬件参数随机生成器
- **功能**:
  - 生成随机 UUID、序列号
  - 支持多种主板品牌
  - 生成真实格式的硬件参数
  - 可保存配置到文件
- **使用**: `./generate-hardware-params.sh`

### 9. generate-hardware-params.bat
- **用途**: Windows 硬件参数随机生成器
- **功能**: 与 .sh 版本相同
- **使用**: 双击运行或 `generate-hardware-params.bat`

## 🎯 使用流程建议

### 新手流程
1. 📖 阅读 `快速上手指南.md` (5分钟)
2. ✅ 运行 `./check-environment.sh` 检查环境
3. 🚀 运行 `./create-windows-vm.sh` 创建虚拟机
4. 🔍 在虚拟机内运行检测脚本验证

### 进阶流程
1. 📖 阅读 `QEMU虚拟机配置指南.md` 了解技术细节
2. 🎲 运行 `./generate-hardware-params.sh` 生成随机参数
3. ✏️ 编辑启动脚本，替换硬件参数
4. 🚀 启动虚拟机
5. 🔍 运行检测脚本验证配置

### 日常使用
1. 🛠️ 使用 `./vm-manager.sh` 管理虚拟机
2. 📊 查看 `速查表.md` 查询命令
3. 🔧 遇到问题查看 `README.md` 故障排查部分

## 💡 使用建议

3. 定期创建快照备份

### 问题排查
1. 查看 `README.md` 的故障排查部分
2. 查看 `QEMU虚拟机配置指南.md` 的常见问题
3. 运行检测脚本找出具体问题

## 📝 文件权限设置

**Linux/macOS 用户需要设置执行权限：**

```bash
# 一次性设置所有脚本权限
chmod +x *.sh

# 或单独设置
chmod +x create-windows-vm.sh
chmod +x create-linux-vm.sh
chmod +x generate-hardware-params.sh
chmod +x test-vm-detection-linux.sh
chmod +x vm-manager.sh
chmod +x check-environment.sh
```

**Windows 用户：**
- `.bat` 文件可直接双击运行
- `.ps1` 文件需要在 PowerShell 中运行
